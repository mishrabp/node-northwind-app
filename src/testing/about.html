About Page

Northwind

This is a learning application built on NodeJS technology stack listed below:

Hosting Platform

Azure

Application is hosted on Microsoft Azure cloud.
Github

Github is being used as the Source Control Management.

Development Technologies

NodeJS

NodeJS provides the runtime environment for Javascript program execution outside browser. The learning app uses NodeJS runtime.
NodeJS

ExpressJS provides the MVC web development framework on top of NodeJS runtime
ejs

EJS is the view engine that works in conjuction with express to build user interfaces.
MongodDB

Is the no SQL database. It stores data in JSON format that Javascript can easily interpret. So there is no need for ORM. No NRM means better performance.
vuejs

VueJS is the MVVM framework enabler that makes your website reactive.
Tailwind

Is the utility based styling. It provides lightweight classes to build responsive UI.

Packaging Tools

Babel

Babel is the transpiler that converts the higher version javascripts to vanilla version so that most of the browser can intepret them.
Gulp

Gulp is the task runner that helps in executing the build tasks everytime there is a change in code.
Browserify

Browserify identifies the module dependency and bundle them.
