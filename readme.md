## General info
This is sample nodejs northwind application built on Nodejs, Tailwind, VueJS, and MongoDB.

* Gulp
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../node-northwind-app
$ npm install
$ npm start
```
